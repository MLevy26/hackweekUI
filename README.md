# HackweekUI

A modern UI for comparing content across different databases including Square Help, Google Drive, and High Spot.

## Features

- Clean, modern interface
- Support for multiple content databases:
  - Square Help
  - Google Drive integration
  - High Spot
- Real-time search functionality for Google Drive
- Responsive design
- Error handling and loading states

## Setup

1. Clone this repository
2. Open `index.html` in a web browser
3. Make sure the backend server is running on `localhost:5000`

## Usage

1. Select your content database from the dropdown
2. For Google Drive:
   - Enter your search topic
   - Click search to see results
3. For other databases:
   - Paste the existing content URL
   - Paste the new content URL to compare

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API for backend communication
